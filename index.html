    Static... what?

    Static sites are the most basic type of website and are the easiest to create. Unlike dynamic websites, they do not require any web programming* or database design.

    A static site can be built by simply creating a few HTML pages and publishing them to a web server.

    * HTML is a markup language; CSS is a style sheet language; and JavaScript is a scripting language.
